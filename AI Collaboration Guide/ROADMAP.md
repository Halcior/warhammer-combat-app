# ROADMAP

## Vision

Build a Warhammer 40k decision-support tool that starts as a solid free combat calculator and grows into a deeper workflow and analysis product.

The business model should be:

- useful free core
- small low-friction premium for convenience and power-user workflows

Not:

- aggressive monetization
- paywalling basic functionality
- overbuilding too early

---

## Phase 0 - Foundation

Goal:

Create a stable and maintainable base.

Key outcomes:

- stable `main`
- fixed broken files and merge leftovers
- cleaner component boundaries
- reliable simulation and calculator flow

---

## Phase 1 - Free MVP

Goal:

Deliver a version that feels like a real tool.

Core features:

- calculator
- weapon/setup comparison
- modifiers and rules
- simulations
- breakdown/explanation
- basic faction support
- good usability

---

## Phase 2 - Premium v1

Goal:

Add low-cost convenience features.

Candidate features:

- saved scenarios
- saved presets
- favorites
- history
- export
- improved workflow features

Pricing should stay low-friction and hobby-friendly.

---

## Phase 3 - Premium v2

Goal:

Become a stronger decision-support tool.

Candidate features:

- roster-aware calculations
- batch testing
- best-target suggestions
- best-weapon suggestions
- matchup workflows
- scenario templates

---

## Phase 4 - Expansion

Possible future directions:

- cloud sync
- shareable links
- community presets
- mobile/PWA improvements
- roster imports
- broader ecosystem features

These are future possibilities, not immediate priorities.
